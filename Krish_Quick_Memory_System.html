🎯 Four Basic Computer Operations

"I PoSt"
Input → Processing → Storage → Output
Yaad rakho: tum jab Instagram pe "I post" karte ho, computer bhi yehi karta hai!

📧 Email Protocols

SMTP = Send Mail To People
POP = Pick Out Posts (retrieve)
SMTP SENDS, POP3 RETRIEVES. Most exams swap karke trick karte hain — yaad rakho!

💾 Storage Sizes (Big to Bigger)

"Boys Kiss Many Girls Till Pregnancy"
Byte → KB → MB → GB → TB → PB
1 KB ≈ 1 thousand bytes, 1 MB ≈ million, 1 GB ≈ billion, 1 TB ≈ trillion, 1 PB ≈ quadrillion

🦠 Malware Types — Quick Difference

VIRUS = Attaches • WORM = Replicates • TROJAN = Disguises
Virus — file ke saath chipakta hai (like glue)
Worm — apni copies banata hai (like rabbits!)
Trojan Horse — achha banke aata hai, andar se bura (movie wala Trojan!)
Spyware — chupke se dekhta hai
Phishing — fake bank email (FISH-ing for your password)

🌐 Network Sizes

LAN → MAN → WAN (chhota → bada)
Local (room/building) → Metropolitan (city) → Wide (country/world)
Internet sabse bada WAN hai!

🔥 Boot Types

COLD = bandh tha 🥶 • WARM = chal raha tha ♨️
Cold boot = power off se start (cold machine garam karna)
Warm boot = restart (already garam hai)

🎨 Drawing vs Painting

DRAW = Vector • PAINT = Bitmap
Drawing programs = Vector (math-based, scalable)
Painting programs = Bitmap/Raster (pixel-based)
Yaad rakho: "DV PB" — Draw Vector, Paint Bitmap

🧠 RAM vs ROM

RAM = "Ram bhool jata hai" 🧠💨
RAM = Random Access Memory = Volatile (power off → data gayab)
ROM = Read Only Memory = Non-volatile (permanent)
RAM ko Ram-jaisa "bhool jaane wala" samjho!

🌟 Network Topologies

Star ⭐ • Bus 🚌 • Ring 💍 • Mesh 🕸️
Star = central hub (like sun)
Bus = ek line (like bus stops)
Ring = circle (data goes round)
Mesh = sab connected (like spider web)

📦 System vs Application Software

System = "Servant" • Application = "Action"
System software = computer ko chalata hai (OS, drivers, utilities) — servant
Application software = user ka kaam karta hai (Word, Excel) — action

🎣 Phishing vs Spoofing

PHISHING = Fishing for info • SPOOFING = Spoof identity
Phishing 🎣 = trusted naam use karke info nikalna (fake bank email)
Spoofing 🎭 = fake sender name dikhana (like fake caller ID)
Exam mein dono confuse karte hain — dhyan se padho!

📡 Modem ka Naam

MOdulator + DEModulator = MODEM
Digital → Analog (sending) = MOdulate
Analog → Digital (receiving) = DEModulate
Phone line analog hai, computer digital — modem dono ke beech translator hai!

🎯 Bridge vs Gateway

Bridge = Same • Gateway = Different
Bridge 🌉 = same type ke networks ko jodta hai (LAN to LAN)
Gateway 🚪 = different networks ko jodta hai (LAN to Internet)
"Gateway = Gate to different world!"

💡 Software Types (License-wise)

Free vs Paid Pyramid
Public Domain = Bilkul free, no copyright
Freeware = Free but copyrighted
Shareware = Try free, then pay
Commercial = Paid software
Open-source = Free + code bhi available (Linux)

🖥️ Computer Sizes (Choti se Badi)

"Mota Mama Mini Mainframe Super"
Microcontroller (chhota chip) → Microcomputer (PC) → Mini (workstation) → Mainframe (badi machine) → Supercomputer (sabse fast)

⚡ CPU ke 2 Parts

"CA + LU = CPU"
Control Unit (CU) = boss, instructions deta hai
ALU (Arithmetic Logic Unit) = worker, math + logic karta hai
Saath mein milke CPU banta hai!

🔌 Serial vs Parallel Port

Serial = Single line • Parallel = 8 together
Serial = ek baar mein 1 bit (queue mein khade log)
Parallel = ek baar mein 8 bits (saath chalte log)
